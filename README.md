# Threat Modeling E-commerce using STRIDE
Threat Modeling de Plataforma E-commerce (STRIDE)

Este repositório apresenta um estudo acadêmico de modelagem de ameaças (Threat Modeling) aplicado a uma arquitetura hipotética de plataforma e-commerce. O trabalho tem como objetivo explorar conceitos de segurança de sistemas e identificar possíveis riscos presentes em aplicações web que lidam com dados sensíveis, como informações de usuários, pedidos e transações.

A análise foi conduzida utilizando o modelo STRIDE, um método amplamente utilizado em engenharia de segurança para classificar ameaças de acordo com seus possíveis impactos. A partir dessa abordagem, foram avaliados diferentes componentes da arquitetura, incluindo navegadores de usuários, servidores web, aplicação e banco de dados.

Durante o estudo, foram consideradas ameaças comuns em ambientes web, como SQL Injection, Cross-Site Scripting (XSS), Cookie Hijacking, ataques de negação de serviço (SYN Flood) e falhas de controle de acesso como IDOR. Cada uma dessas ameaças foi analisada em relação ao seu possível impacto no sistema e classificada dentro das categorias do modelo STRIDE, como Spoofing, Tampering, Information Disclosure e Denial of Service.

Além da identificação das ameaças, o trabalho também discute possíveis estratégias de mitigação, com o objetivo de demonstrar como práticas básicas de segurança podem reduzir riscos em sistemas distribuídos e aplicações web.

Este projeto foi desenvolvido com fins acadêmicos e de aprendizado, buscando exercitar conceitos de segurança da informação e arquitetura segura. A análise apresentada possui caráter introdutório e simplificado, não representando uma avaliação completa de segurança de um sistema real.
