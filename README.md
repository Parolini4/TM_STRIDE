# TM_STRIDE
nálise acadêmica de Threat Modeling em uma plataforma de e-commerce utilizando a metodologia STRIDE. O projeto mapeia a superfície de ataque, identifica ameaças (como DoS, Spoofing e XSS) e propõe medidas de mitigação focadas na defesa da aplicação. Desenvolvido como portfólio de cibersegurança.
